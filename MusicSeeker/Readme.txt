1. Para la ejecución de este software, se debe tener eclipse instalado. 
2. Desde eclipse se importa la carpeta "MusikSeeker".
3. Se edita el archivo "reconocedor.java" y en la inicialización de variable path se le coloca la ruta de la carpeta donde están los archivos .wav.
4. Se ejecuta.
En el archivo "fv.arff" quedan las características extraídas de las canciones.
